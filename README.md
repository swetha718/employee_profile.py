
# Employee Profile Generator

## Overview
A simple Python program that generates and displays employee profile information.

## What It Does
- Creates employee profiles with personal information
- Displays employee details (name, age, position, salary)
- Parses employee codes to extract department, year, and initials
- Formats output in a professional card format

## How to Run
```bash
python employee_profile.py
```
